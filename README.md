# Project TALI

Project TALI, or Thoracic Auxiliary Limb Integration, is a conceptual bio-anatomical design for adding a secondary pair of arms to a humanoid body.

The extra arms emerge from a reinforced upper-to-mid thoracic frame built into the rib cage and spine. This frame distributes force, supports movement, and provides attachment points for new bones, muscles, and nerves.

The design explores how a four-armed humanoid could function through a secondary limb girdle, expanded musculature, a new thoracic nerve plexus, and early developmental adaptation for coordinated control of all four arms.
